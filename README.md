# 🔐 Password Generator Web App

A simple and interactive **Password Generator Web Application** built using HTML, CSS, and JavaScript. This tool helps users generate secure and random passwords with ease, improving password security practices.

👉 **Live Demo:**  
https://uttamshetty-05.github.io/PASSWORD_GENERATOR/

---

## 📌 Features
- Generate strong random passwords  
- Simple and clean user interface  
- Instant password generation  
- Helps improve password security awareness  
- Beginner-friendly frontend project  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

---

## 🎯 Project Objective
This project was developed to:

- Practice JavaScript logic building  
- Understand random password generation  
- Improve frontend development skills  
- Build a useful real-world utility application  

---

## 🚀 How to Run Locally
1. Clone the repository:
   git clone https://github.com/Uttamshetty-05/PASSWORD_GENERATOR.git

2. Open the project folder.

3. Run `index.html` in your browser.
